Description

DetectorJS, A platform, environment, features detector written in ES6 (based on platform.js & feature.js)

Example

Check out this Codepen example

Installation

This is a Node.js module available through the npm registry. It can be installed using the npm or yarn command line tools.

npm install detector-js --save

Dependencies

  • bowser: Lightweight browser detector
  • feature-js: FeatureJS is a fast, simple and lightweight browser feature detection library in 1kb.
  • flavor-js: FlavorJS the definitive JS natives chainable extensions methods (based on lodash & ES6)
  • platform: A platform detection library that works on nearly all JavaScript platforms.
